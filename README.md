# DjangoTasks

## Overview

A simple task app with Django

## Features

- Authentication
- CRUD tasks
- Dark mode

## Built with

- [Django](https://www.djangoproject.com/) - web framework
- [Bootstrap 5](https://getbootstrap.com/) - UI framework
- [Render](https://render.com/) - to deploy
